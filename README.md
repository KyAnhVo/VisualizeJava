# JavaToUML
Your java code doesn't have a UML for it yet? Generate one with JavaToUML!

# Note
This one moved from `https://github.com/KyAnhVo/javaToUml` due to the other one cloning 
too much (String instead of &str and all that jazz)

# Note: Java version 8
The project, as of 0.1.0, only supports java 8.

## Description
This project will capture type relationships and type data for Java projects:
- members and subtypes
- abstraction concepts: inheritance, bounded polymorphism, composition
- metadatas like annotations, etc.
